# Mapgen
 Web Map Generator for OTX Widget MAPGen


------------------------------------------------------------------

Generates Map images for the OpenTX GPS Map widget

https://hobby4life.nl/mapgen/

Change the API key in index.html & process.php to your own API Key for your website.
